Knosy

An AI meeting companion that keeps working after the meeting ends. Your meeting keeps working after you leave.

Perplexity Agent Build Night April 2026 Built in 3 hours Lead dev and design Winner, Best Computer-Powered Product
01Problem

Meetings generate action items and then stall. The notes get written, but no real work starts until someone re-reads them days later.

02Insight
The expensive gap is not capture, it is the cold start on each task.

If the groundwork existed the moment a task was assigned, follow-through would feel almost free.

03What I built

Live transcription on the Web Speech API, automatic action-item extraction with assignee detection, and a head-start engine on the Perplexity Sonar API that researches, drafts, and plans each task before its owner opens it. I scoped, designed, and built it as lead.

Live transcription Auto action items AI head-start
04Decision

Shipped a template fallback for every task type so the demo never failed without an API key. Reliability over raw novelty under a 3-hour clock.

05Result

I won Best Computer-Powered Product. A working live demo, not a mockup.
